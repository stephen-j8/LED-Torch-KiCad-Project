# LED-Torch-KiCad-Project
My first PCB project - LED Torch designed in KiCad 
A Simple 2-layer LED torch circuit designed in KiCad.

## Circuit Details
- Input Voltage Through  Battery 2032
- push button
- 330 ohm Resistor
- LED light

## Tools Used
- KiCad 10.0 (Schematic + PCB Layout )
- JLCPCB (PCB Fabrication - ordered)

## Files Included
- /GERBERS FOR LED TORCH - All fabrication files
- /LED torch SCHEMATIC - PDF Schematic
- /LED torch-LAYOUT - PCB layout PDF
- LED torch.csv - Bill of materials
- TORCH(1) - PHYSICAL PCB AND PCBA
